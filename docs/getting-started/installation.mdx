## Prerequisites

Before you install Motus, make sure you have:

- **Python 3.12 or later**
- **uv** (recommended) or pip
- An API key from at least one LLM provider — OpenAI, Anthropic, Google, or OpenRouter

## Install Motus

<CodeGroup>

```bash uv (from source)
git clone https://github.com/lithosai/motus.git
cd motus
uv sync
```

```bash pip
pip install lithosai-motus
```

</CodeGroup>

<Tip>uv resolves dependencies faster and produces more reliable environments than pip. If you don't have it yet, install it with `curl -Lsf https://astral.sh/uv/install.sh | sh`.</Tip>

### Optional SDK extras

Install optional extras to use Motus alongside other agent frameworks:

<Tabs>
  <Tab title="OpenAI Agents SDK">
    ```bash
    uv sync --extra openai-agents
    ```
  </Tab>
  <Tab title="Claude Agent SDK">
    ```bash
    uv sync --extra claude-agent-sdk
    ```
  </Tab>
  <Tab title="Google ADK">
    ```bash
    uv sync --extra google-adk
    ```
  </Tab>
  <Tab title="All extras">
    ```bash
    uv sync --all-extras
    ```
  </Tab>
</Tabs>

## Verify the installation

Run this one-liner to confirm Motus installed correctly:

```bash
python -c "from motus.agent import ReActAgent; print('Motus is ready')"
```

You should see `Motus is ready` printed to your terminal.

## Set your API keys

Export at least one provider API key before running an agent. Motus reads these from environment variables at runtime.

```bash
export OPENAI_API_KEY=sk-...
export ANTHROPIC_API_KEY=sk-ant-...
export OPENROUTER_API_KEY=sk-or-...
```

<Note>You only need the key for the provider you plan to use. If you are not sure which to pick, OpenAI's `gpt-4o` and Anthropic's `claude-sonnet-4-20250514` are both well-tested with Motus.</Note>
