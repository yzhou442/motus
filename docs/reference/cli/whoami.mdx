---
title: "motus whoami"
description: "Print the identity associated with your current Motus Cloud credentials."
---

Print the identity associated with your current credentials. Use this to confirm which account your CLI is authenticated.

## Usage

```bash
motus whoami
```

## Examples

### Check current identity

```bash
motus whoami
API URL: https://api.lithosai.cloud
API key: sk-1234abcd...
```

### When not logged in

```bash
motus whoami
Not logged in. Run: motus login --api-url <URL>
```
